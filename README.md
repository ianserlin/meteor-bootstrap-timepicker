[Bootstrap Timepicker](http://jdewit.github.com/bootstrap-timepicker/) packaged for Meteor.

###How to use?

1. Install [meteorite](https://github.com/oortcloud/meteorite)
2. `mrt add bootstrap-3-timepicker`
